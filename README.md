# Social-media-app

## install packages
### 1.in root directory 
> npm install 
### 2. change to client directory by cd client
> npm install

## create .env file in server folder
>1. MONGO_URL = "" // this is the URL created from mongodb atlas database
>2. PORT = 3001
>3. JWT_SECRET = "superstrongpassword or anything"

## Start the App
### In two terminal
### 1. cd client
>npm run start
### 2. cd server
>npm start
